# versions ディレクトリについて

## 概要

このディレクトリには、`spec-code-ai-mapper` の各バージョンのスナップショットを保存します。

## ディレクトリ構成

```
versions/
├── v0.8.0/    # 最新版 (AI Mapper)
├── latest -> v0.8.0
└── README.md  # このファイル
```

> [!NOTE]
> v0.7.0 以前のディレクトリは、AI Mapper への移行とプロジェクトの整理に伴い削除されました。

## 更新履歴 (主要なマイルストーン)

### v0.8.0 (2026-02-06)

- **AI Mapper へのリブランディング**: 目的を「設計書-コード紐付けとトレーサビリティ管理」にシフト。
- **セマンティック分割レビュー機能**: md2map/code2mapを使用して大規模ファイルを意味のある単位で分割してレビュー。
- **SharedStateContext 導入**: レビュー機能とマッピング機能の間で状態を共有可能に。

### v0.7.0 (2026-01-25) ※ソース削除済み

- **AIでMarkdownを整理する機能**: AIによる構造化・正規化機能の追加。

### v0.6.0 (2026-01-18) ※ソース削除済み

- **フロントエンド刷新**: Vite + React + TypeScript への移行。
